# Stock-market-forecasting
A Streamlit web app that fetches live stock data from Yahoo Finance, runs ADF test + seasonal decomposition, and forecasts future prices using SARIMAX. Supports any global stock with automatic currency detection (₹, $, €, £). Includes a hidden admin panel for monitoring user searches.
